The supported keywords in template:
  1. ${classname} = class name (in PascalCase format)
  2. ${property.type} = property data type
  3. ${property.name} = property name, by default property name is in camel case format

Note*: ${property.type} and ${property.name} MUST be in loop of properties, as bellow

#foreach($property in $properties)
     ${property.type} ${property.name}
#end

There is a NameCaseUtil that you can use to convert ${classname}/${property.type}/${property.name} to another name case, as bellow:

  • $NameCaseUtil.snakeCase(${property.name}): convert to snake case
  • $NameCaseUtil.camelCase(${property.name}): convert to camel case
  • $NameCaseUtil.kebabCase(${property.name}): convert to kebab case
  • $NameCaseUtil.pascalCase(${property.name}): convert to pascal case
  • $NameCaseUtil.constantCase(${property.name}): convert to constant case
  • $NameCaseUtil.upperCase(${property.name}): convert to upper case
  • $NameCaseUtil.lowerCase(${property.name}): convert to lower case
